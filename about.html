About This Website

Learn more about the purpose and creation of Healthy Living Awareness.

Project Purpose

This website was developed as an individual academic project to demonstrate proficiency in web design and development using standard technologies. The goal is to promote awareness of healthy living by providing simple, reliable information on key areas of wellness: nutrition, exercise, mental health, and sleep.

All content is educational and based on widely accepted health guidelines. It is intended to encourage positive lifestyle habits and is not a substitute for professional medical advice.

About the Developer

Hello! My name is Mensah Lawson Addae , and I am a student currently studying web development and related technologies. This project was created entirely by me using manual coding techniques with HTML5, CSS3, and vanilla JavaScript – no WYSIWYG tools were used.

I chose the topic of healthy living because maintaining balance in physical and mental health is important for everyone. I hope this site provides useful tips and motivates visitors to make small, positive changes in their daily lives.

Technologies Used

  • HTML5 for structure and semantics
  • CSS3 for styling and responsive design
  • Vanilla JavaScript for mobile menu and form validation
  • Royalty-free images from public sources
  • Hosted on a free public platform for accessibility
